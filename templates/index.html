Purchase Request (PR)

จัดการใบขอซื้อทั้งหมด

รายการ PR ทั้งหมด
เลขที่ PR หัวข้อ ผู้ขอ วันที่สร้าง มูลค่า ลำดับอนุมัติ สถานะ จัดการ

Purchase Order (PO)

จัดการใบสั่งซื้อ — สร้าง PO ได้เฉพาะ PR ที่อนุมัติครบทุกลำดับแล้วเท่านั้น

รายการ PO ทั้งหมด
เลขที่ PO อ้างอิง PR ผู้ขาย มูลค่า วันครบกำหนด สถานะ การจ่ายเงิน จัดการ

🚧 โครงการ / Projects

จัดการโครงการก่อสร้าง — เชื่อมข้อมูลกับแผนก

รายการโครงการ
รหัสชื่อโครงการแผนกรับผิดชอบผู้จัดการงบประมาณใช้ไปคงเหลือระยะเวลาสถานะจัดการ

💰 งบประมาณ / Budget

จัดการงบประมาณแยกตามแผนก — เชื่อมข้อมูลจากแผนกอัตโนมัติ

งบประมาณแยกตามแผนก
แผนกงบประมาณใช้ไปคงเหลือการใช้งบสถานะจัดการ

🏢 ผู้ขาย / Vendors

จัดการข้อมูลผู้ขายและคู่ค้า

รายชื่อผู้ขาย
รหัสชื่อบริษัทเลขผู้เสียภาษีผู้ติดต่อเบอร์โทรอีเมลจัดการ

🏗️ จัดการแผนก / Departments

โครงสร้างองค์กร — บริษัทก่อสร้างและวิศวกรรม

โครงสร้างแผนก
รหัสแผนก / ฝ่ายชื่อภาษาอังกฤษประเภทCost Centerอีเมลสถานะจัดการ

👤 จัดการผู้ใช้งาน / Users

จัดการบัญชีผู้ใช้ สิทธิ์การเข้าถึง และรหัสผ่าน

รายชื่อผู้ใช้งาน
รหัสชื่อ-นามสกุลUsernameแผนกตำแหน่งสิทธิ์สถานะจัดการ

⚖️ เปรียบเทียบราคา / Price Comparison

เปรียบเทียบราคาจากผู้ขายหลายราย ก่อนตัดสินใจสร้าง PO — เชื่อมกับ PR ที่อนุมัติแล้ว

📋 สร้างใบเปรียบเทียบราคา
